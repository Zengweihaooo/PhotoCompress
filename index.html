访问iCloud照片指南

🚀 最简单：AirDrop (推荐)

  1. 在iPhone上打开照片应用
  2. 选择有GPS信息的照片 (通常是最近拍的)
  3. 点击分享按钮 → 选择AirDrop → 选择你的Mac
  4. 照片会自动保存到Mac的下载文件夹
  5. 回到这里上传下载文件夹中的照片
小贴士：选择相机拍摄时间前后几小时内的iPhone照片效果最佳

🌐 网页版：iCloud.com

  1. 在浏览器中打开 iCloud.com
  2. 登录你的Apple ID
  3. 点击"照片"
  4. 选择最近的照片 → 点击下载按钮
  5. 回到这里上传下载的照片

⚡ 自动同步 (如果已开启)

如果你的Mac已经开启了iCloud照片同步：

  1. 直接在Finder中打开 ~/Pictures
  2. 照片会自动同步到这里
  3. 选择最近的照片文件夹上传
